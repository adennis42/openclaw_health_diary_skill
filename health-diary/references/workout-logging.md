# Workout Logging Reference

## Log Format

Add to today's journal under `## Workouts`:

```
| Time | Exercise | Sets × Reps | Weight | Notes |
|------|----------|-------------|--------|-------|
| 6:00pm | Barbell Squat | 3 × 8 | 185 lbs | Felt strong |
| 6:15pm | Bench Press | 4 × 6 | 225 lbs | New PR |
| 6:30pm | Bent-over Row | 3 × 10 | 135 lbs | |
```

For cardio/activity:
```
| 4:00pm | Disc Golf (18 holes, walking) | — | — | ~4 miles, 2hrs |
| 7:00am | Walking | — | — | 30 min, ~1.5 miles |
```

## Personal Records (PRs)

Track in `memory/health-diary/prs.md`:
```
| Exercise | Weight | Reps | Date |
|----------|--------|------|------|
| Bench Press | 225 lbs | 6 | 2026-03-12 |
| Squat | 315 lbs | 5 | 2026-03-10 |
```

When a user sets a new PR, update this file and call it out in the reply.

## Common Lifts Reference

### Compound Movements (most important to track)
- Squat, Deadlift, Bench Press, Overhead Press, Barbell Row
- Log every session — these are the best strength progress indicators

### Accessory Movements
- Curls, Tricep extensions, Lateral raises, Cable work, Machine work
- Log when mentioned, but less critical to track for PRs

## Workout Observation Guidelines

- Note if the user mentions soreness, fatigue, or feeling strong
- Celebrate PRs enthusiastically — they matter
- If a user skips a workout, no guilt — just log what happened
- For GLP-1 users (Zepbound/Ozempic): note that maintaining protein intake is especially important to preserve muscle during weight loss
